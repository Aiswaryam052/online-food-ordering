# 🍔 Online Food Ordering System

## 📌 Description
This is a web-based Online Food Ordering System built using PHP and MySQL.

## 🚀 Features
- User Registration & Login
- View Food Menu
- Add to Cart
- Place Orders
- Admin Dashboard

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## ⚙️ How to Run
1. Install WAMP Server
2. Place project inside C:\wamp64\www\
3. Open http://localhost/online_food_ordering/
